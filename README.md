# MallMgmt (mallmgmt)

电子商城后台项目

## 安装依赖项

```bash
yarn
# or
npm install
# or
yarn install
```

### 运行项目

```bash
quasar dev
# or
yarn dev
```

### Lint the files

```bash
yarn lint
# or
npm run lint
```

### Format the files

```bash
yarn format
# or
npm run format
```

### Build the app for production

```bash
quasar build
```

### Customize the configuration

See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
